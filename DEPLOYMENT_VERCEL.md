# Vercel Python Flask deployment

1. Install Vercel CLI:
   npm install -g vercel

2. Login to Vercel:
   vercel login

3. Deploy:
   vercel --prod

- Your Flask app is in api/index.py
- requirements.txt is in api/
- vercel.json is in the project root
- Templates and static files are in their original folders

Visit your Vercel dashboard to set up your custom domain (eaglenewz.com) after deployment.
